# 🐍 Appunti di Python per Principianti

Benvenuto nel mio repository! 👋 
Questa è una raccolta di appunti pratici e script eseguibili pensati per chi sta imparando a programmare in **Python partendo da zero**. 

Invece della solita teoria noiosa, ogni file è un vero e proprio programma Python pieno di commenti chiari, esempi pratici e piccoli esperimenti interattivi.

> 🌱 **Work in Progress:** Questo repository è in continuo aggiornamento! Sto aggiungendo nuovi appunti e file man mano che avanzo nel mio percorso di studio.

## 📂 Cosa troverai in questo repository?

Ho diviso gli appunti per argomenti. Ti consiglio di leggerli e provarli in questo ordine:

1. **`bvn.py`** e **`saluto.py`** - Le basi assolute: come far "parlare" il programma (`print`) e come farlo "ascoltare" (`input`).
2. **`variabili.py`** - Le nostre "scatole" di memoria: tipi di dato (`str`, `int`, `float`, `bool`) e come nominarle correttamente.
3. **`operatori.py`** - La matematica, la logica e i confronti in Python (da `+` a `==`, fino ad `and`/`or`).
4. **`stringe.py`** - Come manipolare il testo, unire le parole e usare i metodi (`.upper()`, `.replace()`, ecc.).
5. **`lista.py`** - L'armadio a cassetti: come salvare tanti dati in fila, usare gli indici e modificarli.
6. **`dizionario.py`** - La rubrica telefonica: salvare i dati usando il sistema "Chiave : Valore".
7. **`if_controllo_di_flusso.py`** - Prendere decisioni con il codice usando `if`, `elif` ed `else`.
8. **`ciclo.py`** - Come ripetere le azioni all'infinito o scorrere gli elementi usando `for` e `while`.
9. **`funzioni.py`** - I "mini-programmi" riutilizzabili: come creare le tue ricette con `def`, usare i parametri e capire `return`.
10. **`importazioni_e_main.py`** e **`usa_importazioni.py`** - Come dividere un progetto grande in più file, usare i moduli (come `math` e `random`) e il trucco di `if __name__ == "__main__":`.

## 🚀 Come usare questi appunti

Puoi semplicemente aprire i file e leggerli come se fossero un libro di testo, oppure (ancora meglio!) puoi eseguirli nel tuo terminale per vedere cosa succede.

Assicurati di avere Python installato sul tuo sistema, poi apri il terminale e digita:

```bash
python3 nome_del_file.py
